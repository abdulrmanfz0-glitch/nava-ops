### Accessing the Dashboard

1. Navigate to **Dashboard v2** from the main menu

- Search or filter by category
- Click any widget to add it to your dashboard

**Step 3: Arrange Your Dashboard**
- In customize mode, drag widgets to reorder them
- Your layout saves automatically
- Click **"Done"** when finished

## ðŸ’¡ Tips & Tricks

### Productivity Tips

1. **Start Your Day with Insights**
   - Check Smart Insights first thing
   - Address high-impact items immediately
   - Set reminders for medium-impact actions

2. **Use Drill-Down for Investigation**
   - Click KPI cards to see detailed breakdowns
   - Compare periods to understand trends
   - Look for patterns in sparklines

3. **Customize by Task**
   - Create different layouts for different needs
   - Export your current layout before experimenting
   - Use presets as starting points, then customize

4. **Leverage Trends**
   - Watch sparklines daily for early warnings
   - Green trends = maintain course
   - Red trends = investigate immediately

### Performance Tips

1. **Limit Widgets**
   - More widgets = slower load times
   - Focus on what you actually use
   - Remove widgets you rarely check

2. **Appropriate Date Ranges**
   - Last 30 days is usually sufficient
   - Longer ranges take longer to load
   - Use shorter ranges for real-time monitoring

3. **Close Unused Tabs**
   - Multiple NAVA tabs can slow performance
   - Use one tab for dashboard
   - Close when not actively using

### Insights Interpretation

1. **Check Confidence Scores**
   - >90% = very reliable, act confidently
   - 70-90% = reliable, verify if critical
   - <70% = suggestion only, investigate further

2. **Consider Context**
   - Seasonal variations are normal
   - External factors (holidays, events)
   - Compare to historical data

3. **Act on Patterns, Not Noise**
   - One bad day isn't a trend
   - Look for consistent patterns over time
   - Use multiple data points to confirm

## ðŸ› Troubleshooting

### Dashboard Won't Load

**Solutions:**
1. Refresh the page (Ctrl+R / Cmd+R)
2. Clear browser cache
3. Check internet connection
4. Try incognito/private mode
5. Contact support if issue persists

### Widgets Showing "No Data"

**Possible Causes:**
- No data for selected date range
- Branch has no activity
- Data still loading (wait 10-15 seconds)

**Solutions:**
1. Expand date range
2. Select "All Branches"
3. Click widget refresh icon
4. Check if branch is active

### Slow Performance

**Solutions:**
1. Reduce number of widgets
2. Use shorter date ranges
3. Close other browser tabs
4. Clear browser cache
5. Use modern browser (Chrome, Firefox, Edge)

### Export Not Working

**Solutions:**
1. Disable popup blockers
2. Check browser download permissions
3. Ensure enough disk space
4. Try different browser
5. Wait for all widgets to load first

### Layout Not Saving

**Solutions:**
1. Check that you're logged in
2. Don't use incognito/private mode
3. Enable browser localStorage
4. Clear site data and re-login
5. Check browser console for errors
